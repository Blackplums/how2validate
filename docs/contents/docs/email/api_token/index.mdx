## Generating an API Token

<Step>
  <StepItem title="Navigate to APIToken Dashboard">
    Go to [How2Validate APIToken Dashboard](https://how2validate.vercel.app/apitoken)
    ![Banner](/images/apitoken/apitoken-1.png "APIToken Dashboard Login")

  </StepItem>

  <StepItem title="Sign-In via GitHub:">
    Use your GitHub account to securely authenticate.
    ![Banner](/images/apitoken/apitoken-2.png "APIToken GitHub Login")
  </StepItem>

  <StepItem title="Dashboard:">
    ![Banner](/images/apitoken/apitoken-3.png "How2Validate APIToken Dashboard")
  </StepItem>

  <StepItem title="Create a New Token:">
    - Click "Create New Token"
    ![Banner](/images/apitoken/apitoken-5.png "How2Validate Create APIToken")

    - Fill in:
      - A unique **email address** where reports will be sent.
      - A **token name** (for your reference, like personal-use, team-A-token, etc.)
    ![Banner](/images/apitoken/apitoken-6.png "How2Validate APIToken details")

  </StepItem>

  <StepItem title="Copy Your Token:">
    - Once created, the token will be shown only once.
    - Copy and store it securely – you won’t be able to see it again.
    ![Banner](/images/apitoken/apitoken-7.png "How2Validate APIToken view")
  </StepItem>
</Step>
